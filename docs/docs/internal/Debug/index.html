概要

デバッグモジュールは、カーネルがクラッシュした時またはカーネルテスト用に解析出来るよう、カーネルログのビデオ出力、メモリ出力を行う。

サブモジュール構成

デバッグモジュールのサブモジュール構成について以下に示す。



サブモジュール構成図

サブモジュール概要
# サブモジュール名 概要
和名 コード名
1 メイン Debug デバッグモジュールの共通的な処理を行う。
2 VRAM出力 DebugVram ビデオにログ出力を行う。
3 メモリ出力 DebugMem メモリにログ出力を行う。
