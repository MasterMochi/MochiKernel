初期化

各サブモジュールの初期化を行う。シーケンスを以下に示す。



初期化シーケンス

ページング管理サブモジュールは、物理メモリ領域管理機能を使用するため、物理メモリ領域管理サブモジュールの後に初期化する。I/Oメモリ領域管理と仮想メモリ領域管理サブモジュールは、ページングを有効にしたヒープ領域で管理するため、ヒープ管理とページング管理サブモジュールの後に初期化する。
